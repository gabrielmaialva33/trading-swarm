## :package: Installation

### Prerequisites

- **[Elixir](https://elixir-lang.org/install.html)** 1.15+
- **[Erlang/OTP](https://www.erlang.org/downloads)** 26+
- **[PostgreSQL](https://www.postgresql.org/download/)** 14+
- **[Git](https://git-scm.com/)**
- **NVIDIA API Key** for AI model access
- **OpenAI API Key** (optional) for GPT integration

### Quick Start

1. **Clone the repository**

```bash
git clone https://github.com/gabrielmaialva33/trading-swarm.git
cd trading-swarm
```

2. **Install dependencies**

```bash
mix deps.get
```

3. **Setup database**

```bash
mix ecto.setup
```

4. **Configure environment variables**

```bash
# Create .env file
cp .env.example .env

# Edit with your API keys
export NVIDIA_API_KEY="your-nvidia-api-key"
export OPENAI_API_KEY="your-openai-api-key"
export DATABASE_URL="postgresql://user:password@localhost/trading_swarm_dev"
```

5. **Setup assets**

```bash
mix assets.setup
mix assets.build
```

6. **Start the Phoenix server**

```bash
mix phx.server
```

7. **Access the application**

Open [http://localhost:4000](http://localhost:4000) in your browser.
